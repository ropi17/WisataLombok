# Website Wisata Lombok (Static Version)

Website ini adalah versi statis yang tidak memerlukan server backend (Node.js). Sangat cocok untuk tugas kuliah atau hosting gratis di GitHub Pages.

## Cara Menggunakan
1.  Cukup klik 2x pada file **`index.html`**.
2.  Website akan terbuka di browser.
3.  Jika diminta Login, silakan klik tombol **Register** untuk membuat akun (akun tersimpan di browser Anda).

## Fitur
-   15 Destinasi Wisata Lombok.
-   Integrasi Google Maps.
-   Multi-bahasa (Ganti bahasa lewat dropdown).
-   Login/Register (Simulasi LocalStorage).

## Deployment (GitHub Pages)
Upload semua file ini ke GitHub, lalu aktifkan fitur GitHub Pages di menu Settings.
